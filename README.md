# Online Course Portal 🎓

A simple and efficient web application built with **React** and **Vite** for managing and browsing courses. This project was developed as part of the **AWT Practical (PAC)**.

## 🚀 Features

- **Home Page**: A centered, welcoming landing page.
- **Course List**: Browse all available courses with categories and difficulty levels.
- **Filtering & Sorting**: Filter courses by category/level and sort them alphabetically.
- **Course Details**: Detailed information for every course, including instructor bios and recommendations.
- **Instructor Profiles**: View expert instructors and the courses they teach.
- **Interested & Enrolled List**: Track courses you are interested in or have joined.
- **Responsive Design**: Clean and simple UI that works on both desktop and mobile.

## 🛠️ Tech Stack

- **Frontend**: React.js
- **Build Tool**: Vite
- **Routing**: React Router DOM
- **State Management**: React Context API
- **Styling**: Vanila CSS (Minimalist Design)

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16.0 or higher)
- npm or yarn

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/jd-thakrar/AWT-PAC.git
   cd AWT-PAC
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```

4. **Open in browser**:
   Navigate to `http://localhost:5173` (or the port shown in your terminal).
